SETUP:
1. run rosdep install --from-paths src --ignore-src -r -y

2. launch: ros2 launch leg_assembly display.launch.py

3. move group node launch :
     on separate terminal, first run "ros2 run joy joy_node"


    then, after sourcing workspace, run "ros2 run teleop_collision teleop_collision"


:D
Architecture:

folder leg_assembly: 

     -contains urdf, meshes and launch files

folder leg_assembly_moveit_config:

     -generated by moveit with the urdf as input, contains controller yaml files and more config files for ros2_control, gazebo etc

folder teleop_collision:

     -contains a node that operates the polarix arm using a joystick, while constantly checking self collision for future states, in which case trajectory is stopped just before collision.
